Two suggestions assuming everything is wired up correctly
1. Set your GPIO input pin to be pulled down. That may prevent any false readings when nothing is connected.
2. Add some bigger delays in there as the PIR (depending on how it's jumpered) will output a detection for a few seconds after the motion is gone. Here is the code I use that works. Of course you need to change the pin numbers. I used this just to test mine when I first got it.

Code: Select all

import RPi.GPIO as GPIO
import time
GPIO.setmode(GPIO.BCM)
GPIO.setwarnings(False)

pir_sensor = 21 #GPIO pin 21
led = 17 #GPIO pin 17

GPIO.setup(pir_sensor, GPIO.IN, GPIO.PUD_DOWN)

current_state = 0
GPIO.setup(led,GPIO.OUT)

while True:
    try:
        time.sleep(0.1)
        current_state = GPIO.input(pir_sensor)
        if current_state == 1:
          print("GPIO pin %s is %s" % (pir_sensor, current_state)) # motion detected
          GPIO.output(led,True) #Turn on LED
          time.sleep(2) # leave LED on for 2 seconds
          GPIO.output(led,False) #turn off LED
          time.sleep(4) # wait 4 seconds for PIR to reset. 
    except KeyboardInterrupt:
        GPIO.cleanup()
